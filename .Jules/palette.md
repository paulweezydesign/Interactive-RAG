# Palette's Journal

## 2024-05-23 - Initial UX Audit
**Learning:** The Streamlit app lacks a clear title in the browser tab and doesn't provide any on-page guidance for its capabilities (like adding URLs or searching the web), which are only discoverable through trial and error or reading the code.
**Action:** Use `st.set_page_config` to set a proper page title and add a sidebar or help section to guide the user.
